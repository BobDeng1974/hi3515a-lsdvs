一. 目录说明
common: 基本接口、基本类的封装,适用于任何一个工程，其他工程
public: 公用接口,仅适用于本工程
lsdvs: 本工程的所有模块代码
mppinc: 该目录只是为了查看方便而添加的,都是海思mpi 的头文件。

二. 编译指导
    该工程是linux 跨平台的，根据需求可以再海思或者TI的嵌入式开发板上运行，需要编译的时候添加参数。
1. 编译用于海思3515开发板运行的方法：在lsdvs 目录下运行make HI3515=1
2. 编译用于TI MA3359开发板运行的方法：在lsdvs 目录下运行make MA3359=1
3. 清除编译: make cleanall
3. 建立依赖: make HI3515A=1 dep; 


特别注意: 当使用make cleanall 
	  指令后,所有的依赖关系也被清除掉，需要重新执行make HI3515A=1 dep 来建立依赖关系。(现在只做了HI3515A,未进行其他平台的开发)
